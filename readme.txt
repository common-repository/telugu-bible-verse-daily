Lets you display either the verse of the day from telugubibleonline.com or a random verse from your favorites. 

== Description ==
తెలుగు బైబిల్ వచనములు allows you to display a bible verse on your site either in posts/pages or via the included widget. 

* You can have it display a verse randomly from your favorites, or display telugubibleonline.com's Verse of the Day
* Verses can be displayed only in Telugu language
* Widget included (You can drag and drop the widget to any place from Widgets section)

== Installation ==
* Install directly from WordPress (go to Plugins -> Add New)
OR
* Install manually
1. Unzip the plugin files to the `/wp-content/plugins/telugubibleversedisplay/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set your preferred settings through the తెలుగు బైబిల్ వచనములు  section under WordPress' Settings menu
